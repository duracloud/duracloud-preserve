# External documentation
Documentation provided by third-party clients recommended for providing access to your account's managed S3 buckets:

- [Amazon command line](https://aws.amazon.com/cli/) documentation
- [Cyberduck documentation](https://docs.cyberduck.io/cyberduck/), including for downloading and [installing their client](https://docs.cyberduck.io/cyberduck/installation/) on Mac or Windows devices
- [SFTPGo documentation](https://docs.sftpgo.com/enterprise/) (mostly intended for institutions setting up their own infrastructure)
